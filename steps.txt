# GeoDrift Impact Analysis Tool

## Overview
The **GeoDrift Impact Analysis Tool** is a web-based application designed to visualize and analyze land-use changes over time. It allows users to upload "Before" and "After" shapefiles, perform client-side change detection, and generate detailed reports and visualizations of the transformations.

## Features & Functionality

### 1. Data Upload
*   **Dual Drop Zones**: The interface provides two distinct areas for uploading data:
    *   **Before State**: For the baseline dataset (e.g., year 2016).
    *   **After State**: For the comparison dataset (e.g., year 2022).
*   **File Support**: Accepts **.zip** files containing standard Shapefile components (`.shp`, `.shx`, `.dbf`).
*   **Automatic Year Detection**: The tool automatically detects the year from the filename (e.g., "file_16.zip" -> 2016) and updates the interface labels accordingly.
*   **Visual Feedback**: Uploaded layers are immediately rendered on the map with a default style.

### 2. Attribute Selection & Symbology
*   **Attribute Dropdown**: Once files are uploaded, a dropdown menu populates with available attribute columns from the shapefile data.
*   **Dynamic Styling**: Selecting an attribute (e.g., "LandUse") automatically applies a unique color palette to both maps, allowing for immediate visual comparison of the raw data.
*   ** synchronized Maps**: The "Before" and "After" maps are synchronized by default. Panning or zooming one map automatically updates the other to maintain the same view.

### 3. Change Analysis
*   **Client-Side Processing**: Clicking **"Analyze Changes"** triggers a powerful client-side spatial intersection algorithm (using Turf.js).
*   **Transition Detection**: The tool identifies areas where the selected attribute has changed between the two time periods (e.g., "Open Land" → "Built Up").
*   **Optimization**: Adjacent polygons with the same transition type are dissolved (merged) to create a cleaner map and reduce visual clutter.

### 4. Visualization Results
*   **Change Map**: A dedicated view displays the results:
    *   **Unchanged Areas**: Shown in **Green**.
    *   **Changed Areas**: Assigned unique colors based on the specific transition type (e.g., Red for "Forest -> Urban").
*   **Interactive Legend**:
    *   Displays all transition types with their corresponding colors.
    *   **Click to Highlight**: Clicking a legend item highlights only those specific changes on the map, dimming others.
    *   **View All**: Restores visibility of all features.
*   **Smart Tooltips**: Permanent labels are automatically placed on the largest polygon of each transition type to provide context without overcrowding the map.

### 5. Statistics & Insights
*   **Summary Stats**: A panel displays key metrics:
    *   Total Analyzed Area (in Hectares).
    *   Total Changed Area & Percentage.
    *   Unchanged Area.
*   **Top Changes**: A list of the most significant land-use transitions ranked by area size.

### 6. Export & Reporting
*   **Download Map Image**:
    *   Generates a high-quality PNG of the current map view.
    *   **Smart Capture**: Automatically hides tooltips and adds a professional legend overlay with year information and area statistics directly onto the image.
    *   **Auto-Zoom**: Ensures the map is perfectly zoomed to fit the extent of the data before capturing.
*   **Download Report**: Creates a standalone HTML report containing the map image, summary statistics, and a detailed breakdown of changes, ready for sharing.

## Usage Guide
1.  **Upload**: Drag and drop your `.zip` shapefiles into the "Before" and "After" boxes.
2.  **Select Attribute**: Choose the column representing land use (e.g., `class_name`) from the dropdown.
3.  **Analyze**: Click the "Analyze Changes" button.
4.  **Explore**: Use the interactive legend to filter changes and view statistics in the side panel.
5.  **Export**: Use the download buttons to save a map image or a full report.
